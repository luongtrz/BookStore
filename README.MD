Node.js (ver >= 14.x)
Vercel

# Backend: 
cd app
npm install

host in port 5000
or npm run dev (nodemon)

GET /api/books: Get all books
GET /api/books/:id: Get a book by ID
GET /api/genres: Get all genres
POST /api/users/register: Register a new user

# Frontend: 
cd ../view
npm install
npm run dev / npm run build // host in port 3000
